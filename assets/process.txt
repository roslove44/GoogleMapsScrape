=== v1 - 05/07/2023 - Comment je scrape Google Maps ===

pour avoir ce que je cherche :
1- aller sur le site : https://www.google.com/maps/
2- taper : boulangerie cotonou dans la barre de recherche ayant pour id searchboxinput
3- press enter
4- attendre que la page se charge complètement
5- atteindre le conteneur de résultats :
   body -> div#app-container -> div#content-container -> div#QA0Szd
   il faut scroller dans cette section pour que les résultats se chargent

   chemin vers le feed :
   -> div.XltNde.tTVLSc -> div.w6VYqd -> div.bJzME.tTVLSc
   -> div.k7jAl.lJ3Kh.miFGmb -> div.e07Vkf.kA9KIf -> div.aIFcqe
   -> div.m6QErb.WNBkOb[role="main"]
   -> div.m6QErb.DxyBCb.kA9KIf.dS8AEf.ecceSd[aria-label="Résultats pour ..."]

6- scroller jusqu'à ce que div.m6QErb.tLjsW.eKbjU apparaisse (marqueur de fin de liste)

sélecteurs que j'utilise pour les infos de chaque card :
- activité : div > div.bfdHYd ... > span
- site web : div.lI9IFe > div.Rwjeuc > div:nth-child(1) > a
- téléphone : div.UaQhfb ... > span:last-child > span:last-child
- lien fiche : div > a

quand la recherche tombe sur un autre pays :
- #omnibox-directions > div > div:nth-child(2) > div > button

snippet js pour tester le feed dans la console :
var element = document.querySelector("div.m6QErb.DxyBCb.kA9KIf.dS8AEf.ecceSd");
element ? element.style.backgroundColor = "red" : console.log(false);

snippet js pour tester le marqueur de fin :
var elementx = document.querySelector("div.m6QErb.tLjsW.eKbjU");
elementx ? elementx.style.backgroundColor = "red" : console.log(false);


=== v2 - 12/03/2026 - Mise à jour après changements Google Maps ===

google a changé des trucs, je dois revoir ma logique de détection de page.

je me suis rendu compte que je peux tout détecter avec div.m6QErb.XiKgde[role="region"] :
- 0 regions → another_country ou rien → skip
- 1 region → zone géographique (genre "Cotonou" tout court) → skip
- 2+ regions → fiche unique (single) → je parse
- div[role="feed"] présent → liste d'entreprises → scroll + parse les cards

pour le cas "another_country" j'ajoute aussi :
#omnibox-directions, #directions-searchbox-0, #directions-searchbox-1

pour le single, je parse la fiche directement :
- titre section (div.tAiQdd) :
  - nom : h1.DUwDvf.lfPIob
  - note : div.F7nice span[aria-hidden="true"]
  - avis : 2e span[role="img"] dans div.F7nice
  - activité : button.DkEaL
- détails section (2e div.m6QErb.XiKgde[role="region"]) :
  - 1er div.Io6YTe.fontBodyMedium.kR99db → adresse
  - les suivants → téléphone (is_valid_phone_number) et site web (is_valid_domain)

pour le scroll du feed :
- j'utilise div.m6QErb.tLjsW.eKbjU comme condition d'arrêt (pas le scrollHeight)
- scrollBy(0, 400) avec 1s de pause entre chaque scroll
- google limite à ~120 résultats par recherche, conseiller à l'user de découper par quartier
